01 — 设计理念 Philosophy

紫为体，火为用

品牌名取自三元九运的「九紫离火运」。设计上把它拆成两股力：九紫是沉稳、神秘、高级感的主色（主按钮、链接、焦点环、品牌识别）；离火是行动与能量的信号（充值/开通等关键转化、强调辉光、数据高光）。二者之间用一条「紫→绛→焰→金」的信号渐变贯穿，是整套语言的灵魂。

深色优先
玄夜近黑、微偏紫的画布让离火的辉光真正「燃」起来；浅色作为同源对偶提供给偏好亮色的用户。
克制的能量
渐变与辉光只用在品牌时刻与关键转化上，数据密集界面保持冷静，避免「玄学」沦为花哨。
与 new-api 同源
全部令牌沿用现有 shadcn / Tailwind v4 变量名，作为一个 theme-preset 落地，可随上游持续更新。

07 — 落地映射 Implementation

如何接入 web/default

这套语言不是另起炉灶，而是作为一个 theme-preset 接入现有体系，因此能随上游 new-api 持续合并更新。两处改动即可点亮品牌：

① 注册预设 — src/lib/theme-customization.ts
// 加入 THEME_PRESETS 数组 { value: 'jzlh', name: '九紫离火 · Nine Purple Li Fire', swatches: ['oklch(0.66 0.215 305)', 'oklch(0.64 0.225 30)'] }
② 定义令牌 — src/styles/theme-presets.css
[data-theme-preset='jzlh'] { /* 浅色 */ --primary: oklch(0.515 0.235 305); --secondary: oklch(0.95 0.03 312); --ring: oklch(0.515 0.235 305); --chart-1: …; --chart-2: …; /* 九紫离火谱 */ --radius: 1rem; } .dark [data-theme-preset='jzlh'] { /* 深色（主画布） */ --primary: oklch(0.66 0.215 305); --background: oklch(0.185 0.014 312); /* 玄夜偏紫 */ … }

渐变与辉光（--gradient-lihuo / --glow-*）作为新增品牌变量加在同一处； .btn-cta 等品牌类放进 styles/index.css 的 @layer components。

③ 品牌化收尾（覆盖 web/default 全量时）
public/logo.png + favicon.ico 换新； index.html 的 <title> / theme-color；登录/注册（routes/(auth)）、 落地页（features/home）、定价（features/pricing）的 hero 用离火渐变；侧栏激活态焰点。注意：保留 new-api / QuantumNous 的版权头与开源署名（项目政策保护项，不可移除）。
